# Ghana Tertiary UniConnect Dataset

## Counts
- Institutions: 68
- Faculties: 350
- Departments: 1681
- Courses: 6724
- Regions: 16

## Files
- JSON datasets in `/json`
- CSV datasets in `/csv`
- Supabase seed in `/sql/supabase_seed.sql`
- Firebase seed in `/firebase/firebase_seed.json`

## Important
This is a launch-ready starter dataset for UniConnect, not a final official national academic registry. Institution accreditation and exact programmes must be verified from GTEC and each institution.

Use this for:
- university selection
- faculty/department/course dropdowns
- onboarding
- marketplace categories
- event categories
- election positions
- clubs and associations
