# Ghana Tertiary Dataset SQL Editor Parts

The generated dataset is too large for the Supabase SQL Editor as one query, so it has been split into ordered chunks.

Run these files one by one in Supabase SQL Editor:

1. `part_001.sql`
2. Continue sequentially through the final part.

Do not skip parts. `part_001.sql` contains the schema/table/policy setup, and later parts continue the seed data.

Total parts: 45

If you connect directly with `psql` or Supabase CLI, you can run the same parts in filename order.
